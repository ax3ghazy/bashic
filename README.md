# BAsic SHell In C (BASHIC)


## Running

```
gcc src.c -o bashic
./bashic
```
use `exit` to terminate the shell

## Specs

- `cd`
- Colorized prompt
- Children are sent SIGINT (not SIGKILL) when the shell terminates
